Introduction

This guide will walk you through setting up an empty scene to use the Monster Fly prefab.
The package comes with a demo scene where you can find example of how to use the prefab.

Setup

Let’s create a new scene:
	- Create a new scene
	- Open the prefabs Folder
	- Add the prefabs in the scene
	- Select the "DemoScript"
	- In the "Anim" property add at least one "Monster_Fly Variant" added to the scene 
	- Done
	- Have Fun!
